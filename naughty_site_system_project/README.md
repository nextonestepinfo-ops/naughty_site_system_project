# naughty Webサイト・管理システム構築プロジェクト

作成日: 2026-05-29

## 目的

コンカフェ「ノーティ」のWebサイトと、店舗側で使う簡単な管理システムを並行して作る。

Webサイトは外向けの見せ方を整えるもの。  
管理システムは、スマホ中心でも使えるように、スタッフ・出勤・シフト・商品・売上・給与まわりを簡単に更新できるもの。

## 大事な方針

- 既存Webサイト案のトップ部分は、Vすぽ風に参考にしたファーストビュー要素を必須で活かす。
- その他のページ構成、デザイン、素材の使い方は再検討する。
- Webサイトを本格的なエディター/CMSにはしない。
- 管理システム側に情報を入れたら、Webサイト側のスタッフ紹介、写真、紹介文、出勤、シフト、イベント、メニューに反映される形を目指す。
- コンピューターに強くない人、スマホ中心の人でも直感的に使える画面にする。
- システム側にはテスト商品、テストスタッフ、テストシフトを入れて構築を進める。

## フォルダー構成

- `00_source/`
  - 元ZIPや受領素材を置く場所。
- `01_existing_site/`
  - 既存のWebサイト案を展開した場所。
- `02_docs/`
  - ヒアリング整理、要件、作業順などの資料。
- `03_system_seed/`
  - テストスタッフ、テスト商品、テストシフトなどの初期データ案。
- `04_site_rebuild/`
  - 今後、再構成したWebサイトを作る場所。
- `05_admin_system/`
  - 今後、管理システムを作る場所。
- `99_handoff/`
  - 次チャットや別作業者へ引き継ぐためのメモ。
