# 管理システム作業場所

ここに、今後ローカルで動く管理システムを作る。

## 最初に作る画面案

- ログイン
- 管理者トップ
- スタッフ管理
- スタッフ写真/紹介文編集
- 2週間シフト管理
- 出退勤/勤務ステータス
- 商品マスター
- 売上登録
- CSV出力
- 給与確認

## Webサイト連動

管理システム側のデータを、Webサイト側が読み込める形式で出す。

最初はJSONでもよい。

## 現在のプロトタイプ

- `index.html`
  - スマホ運用を想定したローカル管理画面。
  - ホーム、スタッフ、シフト、メニュー、イベント、売上、給与の初期画面を実装。
- `app.js`
  - `../03_system_seed/naughty_site_data.json` を初期値として読み込む。
  - 保存時に `localStorage` へ反映し、同じローカルURLの公開サイトが読み込めるようにする。
  - 売上CSV出力、15分単位切り捨ての日給計算土台、支払確定時の売上バック集計リセットを含む。
- `styles.css`
  - 大きいボタン、選択式、少ない入力を優先した管理UI。

ローカル確認時はプロジェクトルートでHTTPサーバーを起動し、`/05_admin_system/index.html` を開く。
