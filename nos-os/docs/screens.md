# Nos OS Screen List

## Core Screens

| Path | Screen | Primary Users | Phase1 Scope |
| --- | --- | --- | --- |
| `/login` | Login | All | Google/email demo login with role selection |
| `/` | Dashboard | All | Role-based overview and AI recommendations |
| `/projects` | Projects | All | List, filters, status, budget, owners |
| `/projects/[id]` | Project Detail | All | Project info, tasks, email placeholder, history, files |
| `/tasks` | Tasks | All | List, kanban, calendar-style view, filters, sort |
| `/customers` | Customers | Admin | Client list and project count |
| `/employees` | Employees | Admin | Employee list and status overview |
| `/employees/[id]` | Employee Profile | Admin/self | Profile, skills, projects, stats, leave, goals, AI analysis |
| `/attendance` | Attendance | All | Clock actions, QR placeholder, history, leave balance |
| `/assistant` | AI Assistant | All | Today recommendations and workload/priority advice |
| `/notifications` | Notifications | All | In-app notifications and PWA permission action |
| `/settings` | Settings | Admin/self | Profile, theme, integrations readiness |

## Mobile Navigation

- Bottom tab bar: Dashboard, Projects, Tasks, Attendance, Assistant.
- Secondary actions are available through the top action menu and settings.
- Desktop uses a persistent left sidebar.
